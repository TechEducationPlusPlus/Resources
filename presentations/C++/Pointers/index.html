What are pointers?
Pointer is address, address of variable in the memory.


How to work with them?
```cpp int a = 5; int *ptr = &a; ```
```cpp int f = 5; int *ptr = &f; cout << f << *ptr << "\n"; // values cout << &f << ptr << "\n"; // addresses ```
